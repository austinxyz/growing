# Growing App - 需求概要

## 1. 项目概述

Growing 是一个个人成长知识管理系统，用于管理职业发展、技能提升、面试准备和简历优化。

## 2. 核心功能模块

### 2.1 用户管理模块

- 支持多用户系统
- 用户可以创建和管理自己的账号
- 每个用户独立管理自己的成长数据

### 2.2 成长领域管理

每个用户的成长分为两大类：

1. **职业成长**
   - 软件工程师路径
   - 职业经理路径
   - 其他职业路径（可扩展）

2. **业余爱好成长**
   - 待定（后续扩展）

### 2.3 技能体系

#### 2.3.1 公共技能（跨职业路径）

适用于软件工程师和职业经理的通用技能：

- 编程和数据结构 ⭐（重要技能）
- 系统设计 ⭐（重要技能）
- Behavioral 技能 ⭐（重要技能）

#### 2.3.2 软件工程师专属技能

**细分领域技能**
- 云计算 ⭐（重要技能）
- AI/机器学习 ⭐（重要技能）
- Infrastructure
- DevOps

#### 2.3.3 职业经理专属技能

- 项目管理
- 人员管理 ⭐（重要技能）

#### 2.3.4 技能管理功能

- 技能分类和层级管理
- 技能描述和学习资源
- 每个技能可关联试题库
- 重要技能标记（⭐）并有专门的学习模块

### 2.4 学习模块系统

为重要技能设计专门的学习模块，以下为概要功能，具体细节会在专门模块设计时细化：

1. **编程和数据结构模块**
   - 题目练习
   - 算法分类
   - 解题记录

2. **系统设计模块**
   - 案例学习
   - 设计模式
   - 系统架构

3. **云计算模块**
   - 云服务知识
   - 实践项目

4. **AI/机器学习模块**
   - 算法理论
   - 实战项目

5. **Behavioral 模块**
   - 场景问题
   - STAR 方法练习

6. **人员管理模块**
   - 管理理论
   - 案例分析

### 2.5 公司管理模块

维护目标公司信息：

- **公司基本信息**
  - 公司名称
  - 行业领域
  - 公司规模

- **职位管理**
  - 职位名称
  - 职位要求
  - 关联的重要技能

- **面试准备**
  - 面试经验记录
  - 公司看重的技能清单
  - 公司特定的试题库

### 2.6 简历管理模块

- **简历基础信息**
  - 个人信息
  - 教育背景
  - 工作经历
  - 项目经验
  - 技能清单

- **简历优化**
  - 针对特定公司/职位优化简历
  - 突出目标公司看重的技能
  - 调整项目经验描述
  - 生成定制化简历版本

- **试题库生成**
  - 根据目标职位生成专门的试题库
  - 整合公司特定试题
  - 按技能分类组织

### 2.7 试题库管理

- **试题分类**
  - 按技能分类
  - 按难度分类
  - 按公司分类

- **试题内容**
  - 题目描述
  - 参考答案
  - 解题思路
  - 相关知识点

- **练习记录**
  - 做题历史
  - 答题结果
  - 复习计划

## 3. 用户角色

### 3.1 首个用户（Austin）

- 职业路径：软件工程师 + 职业经理
- 关注技能：
  - 编程和数据结构
  - 系统设计
  - 云计算

  - Behavioral
  - 人员管理

## 4. 系统架构

### 4.1 技术栈

- 后端：Java 17 + Spring Boot 3.2
- 前端：Vue 3
- 数据库：MySQL 8.0

### 4.2 核心实体关系

```
User (用户)
  ├─ CareerPath (职业路径)
  │   └─ Skill (技能)
  │       ├─ Question (试题)
  │       └─ LearningModule (学习模块)
  │
  ├─ Resume (简历)
  │   └─ CustomizedResume (定制简历)
  │
  └─ Company (公司)
      ├─ Position (职位)
      ├─ CompanySkill (看重的技能)
      ├─ InterviewExperience (面试经验)
      └─ CompanyQuestion (公司试题)
```

- **安全性**: 用户数据隔离，访问控制

- **可维护性**: 清晰的代码结构，完善的文档
- **性能**: 快速的查询响应，高效的数据存储

1. **三层架构设计** - 灵活的技能分类体系（Skill → Major Category → Focus Area）
2. **STAR框架答题** - 动态模版系统，支持Behavioral面试准备

7. **响应式卡片设计** - 统一的UI设计语言（学习仪表盘 + 求职仪表盘）
