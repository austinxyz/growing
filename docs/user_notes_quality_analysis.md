public int countNodes(TreeNode root) {
    if (root == null) return 0;

    int leftHeight = getHeight(root, true);
    int rightHeight = getHeight(root, false);

    if (leftHeight == rightHeight) {
        // 满二叉树
        return (1 << leftHeight) - 1; // 2^h - 1
    }

    // 递归计算
    return countNodes(root.left) + countNodes(root.right) + 1;
}

private int getHeight(TreeNode node, boolean goLeft) {
    int height = 0;
    while (node != null) {
        height++;
        node = goLeft ? node.left : node.right;
    }
    return height;
}
```

## 解题思路
DFS遍历每个岛屿，计算面积并标记为已访问，返回最大面积

## 解题步骤
1. 遍历grid的每个单元格
2. 遇到1（陆地）时，调用DFS计算该岛屿面积
3. DFS逻辑：
   - 边界检查：越界或值为0则返回0
   - 标记当前单元格为0（避免重复访问）
   - 递归计算四个方向的面积
   - 返回：1 + 上 + 下 + 左 + 右
4. 更新maxArea = max(maxArea, 当前岛屿面积)
5. 返回maxArea

## 代码示例
```java
public int maxAreaOfIsland(int[][] grid) {
    int maxArea = 0;
    for (int i = 0; i < grid.length; i++) {
        for (int j = 0; j < grid[0].length; j++) {
            if (grid[i][j] == 1) {
                maxArea = Math.max(maxArea, dfs(grid, i, j));
            }
        }
    }
    return maxArea;
}

private int dfs(int[][] grid, int i, int j) {
    if (i < 0 || i >= grid.length || j < 0 || j >= grid[0].length || grid[i][j] == 0) {
        return 0;
    }
    grid[i][j] = 0; // 标记为已访问
    return 1 + dfs(grid, i-1, j) + dfs(grid, i+1, j) +
               dfs(grid, i, j-1) + dfs(grid, i, j+1);
}
```

## 核心策略
DFS计算单个岛屿面积（递归四个方向），遍历所有岛屿取最大值
```
