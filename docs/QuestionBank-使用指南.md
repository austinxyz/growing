# 试题库使用指南

## 功能概述

试题库是一个职业技能试题练习平台，支持：
- 多维度试题搜索和过滤
- 答题模式（支持答题模板）
- 批改模式（对比AI参考答案）
- 答案保存和覆盖管理

## 页面访问

访问路径：`/question-bank`

路由名称：`QuestionBank`

## 功能特性

### 1. 试题搜索和过滤

支持以下过滤条件（级联）：

1. **关键字搜索**
   - 搜索试题标题和描述
   - 支持Enter键快速搜索

2. **职业路径 → 技能 → 大分类 → Focus Area**
   - 级联选择，自动加载下一级选项
   - 可以在任一级别停止选择

3. **试题类型**
   - Behavioral（行为面试题）
   - Technical（技术面试题）
   - Design（设计题）
   - Programming（编程题）

4. **难度级别**
   - Easy
   - Medium
   - Hard

### 2. 两栏布局

#### 左侧：试题列表（宽度：96）
- 显示试题类型、难度、标题
- 已答题的题目显示绿色标记
- 点击选择题目进入答题区

#### 右侧：答题/批改区域
- **答题模式** tab：编辑和提交答案
- **批改模式** tab：查看AI参考答案和对比

### 3. 答题模式

#### 3.1 基本信息展示
- 答题要求（Answer Requirement）
- Red Flags（需要避免的情况）

#### 3.2 答题模板支持
- 自动加载技能关联的答题模板
- 模板包含多个结构化字段（如STAR模板）
- 支持自由答题（无模板时）

#### 3.3 核心策略记录
- 记录答题的核心思路和关键要点
- 便于后续复习和总结

#### 3.4 Markdown预览
- 实时预览答案格式
- 支持Markdown语法

### 4. 批改模式

#### 4.1 AI参考答案
- 查看系统提供的标准答案（如果有）
- 包含核心策略提示

#### 4.2 用户历史答案
- 查看之前保存的答案
- 显示最后更新时间

#### 4.3 当前答案编辑
- 在批改模式下仍可编辑答案
- 支持覆盖之前的答案

#### 4.4 对比视图
- 并排对比AI答案和用户答案
- 快速发现差异和改进点

### 5. 答案保存逻辑

#### 5.1 首次答题
- 点击"保存答案"按钮
- 答案自动保存到数据库

#### 5.2 覆盖保存
- 在批改模式下修改答案
- 点击"覆盖保存"按钮
- 系统提示确认覆盖
- 确认后覆盖之前的答案

#### 5.3 数据保存
保存内容包括：
- 答案内容（Markdown格式）
- 核心策略（可选）
- 自动记录创建/更新时间

## 技术实现

### 前端组件

```
QuestionBank.vue           # 主页面
├── AnswerMode.vue        # 答题模式组件
└── ReviewMode.vue        # 批改模式组件
```

### API接口

#### 搜索试题
```javascript
GET /api/questions/search
参数：
  - keyword: 关键字
  - careerPathId: 职业路径ID
  - skillId: 技能ID
  - majorCategoryId: 大分类ID
  - focusAreaId: Focus Area ID
  - questionType: 试题类型
  - difficulty: 难度
```

#### 获取试题详情
```javascript
GET /api/questions/{id}
返回：试题详情 + 用户笔记 + AI笔记
```

#### 保存/更新笔记
```javascript
POST /api/questions/{id}/note
请求体：
  {
    "noteContent": "答案内容（Markdown）",
    "coreStrategy": "核心策略"
  }
```

### 数据库表

- `questions` - 试题表
- `user_question_notes` - 用户笔记表
  - `user_id = -1` 表示AI笔记
  - `user_id > 0` 表示用户笔记
- `answer_templates` - 答题模板表
- `skill_templates` - 技能-模板关联表

## 使用流程

### 典型答题流程

1. **进入试题库**
   - 点击左侧导航"试题库"

2. **搜索试题**
   - 选择职业路径、技能等筛选条件
   - 或直接搜索关键字
   - 点击"搜索"按钮

3. **选择试题**
   - 从左侧列表选择要练习的试题
   - 右侧自动显示试题详情

4. **答题**
   - 阅读答题要求和Red Flags
   - 如有模板，按模板字段填写
   - 记录核心策略
   - 点击"保存答案"

5. **批改**
   - 切换到"批改模式" tab
   - 查看AI参考答案（如果有）
   - 对比自己的答案
   - 发现不足，修改答案
   - 点击"覆盖保存"（可选）

6. **复习**
   - 已答题的试题会显示绿色标记
   - 可以重新打开查看和修改答案

## 注意事项

1. **答案覆盖**
   - 覆盖操作不可撤销
   - 系统会提示确认
   - 建议在确认改进后再覆盖

2. **模板使用**
   - 模板是建议性的，可以不完全按模板回答
   - 自由答题模式同样有效

3. **核心策略**
   - 建议每道题都记录核心策略
   - 便于快速复习重点

4. **Markdown格式**
   - 支持标准Markdown语法
   - 预览功能可实时查看格式

## 后续改进方向

1. **学习进度跟踪**
   - 统计已答题数量
   - 按难度、类型统计

2. **答题质量评估**
   - AI自动评分
   - 提供改进建议

3. **学习路径推荐**
   - 根据答题情况推荐下一题
   - 智能复习提醒

4. **答案分享**
   - 分享优秀答案给其他用户
   - 学习他人的优秀答案

5. **答题历史**
   - 记录每次修改的版本
   - 可以查看答题进步历程
