## 🧪 测试步骤

### Step 1: 进入求职申请页面

1. 访问 http://localhost:3001
2. 登录后，点击左侧菜单 **"求职管理"**
3. 选择子菜单 **"职位申请"**

### Step 2: 创建或选择职位申请

**如果没有职位申请**：
1. 点击 **"公司职位管理"** 先创建公司和职位
2. 返回 **"职位申请列表"**
3. 为职位创建申请

**如果已有职位申请**：
1. 从左侧列表选择一个职位申请
2. 确保该申请有面试阶段（如果没有，需要先添加面试阶段）

### Step 3: 进入面试阶段

1. 在职位详情页面，点击 **"面试流程"** Tab
2. 从左侧阶段列表选择一个面试阶段（例如 "Phone Screen"）
3. 在右侧面试阶段详情中，确保选中 **"准备笔记"** sub-tab

### Step 4: 查看TODO列表区域

在"准备笔记"tab下方，应该能看到：
- **标题**: "准备TODO清单" （带紫色勾选图标）
- **进度条**: 显示完成进度（初始为0%）
- **新建TODO按钮**: 蓝紫色渐变按钮
- **空状态提示**: "暂无TODO项"

## 🐛 常见问题排查

### 问题1: TODO列表不显示

**可能原因**:
- 后端服务未启动
- 未选择面试阶段
- API调用失败

**排查步骤**:
1. 打开浏览器控制台（F12）
2. 查看Network标签页
3. 检查 `/api/interview-stages/{id}/todos` 请求
4. 如果404，检查后端服务
5. 如果403，检查JWT token

### 问题2: 创建TODO失败

**可能原因**:
- 标题未填写
- 网络错误
- 权限问题

**排查步骤**:
1. 确认标题已填写
2. 查看控制台错误信息
3. 检查Network请求状态码
4. 验证用户已登录

### 问题3: Markdown渲染异常

**可能原因**:
- markdown-it未正确安装

**解决方案**:
```bash
cd frontend
npm install markdown-it
```

### 问题4: 进度条不更新

**可能原因**:
- 组件状态未刷新

**解决方案**:
- 切换到其他tab再切换回来
- 或刷新页面
