# 准备清单功能使用指南

## 功能概述

在"求职-面试-申请列表"页面的"准备材料"tab中，可以为每个面试阶段管理准备清单，支持：
- ✅ **编辑清单项**：修改内容、分类、优先级
- ✅ **添加链接**：支持Markdown格式，可链接到LeetCode、文档等
- ✅ **重点标记**：标记为重点的项目会单独显示，方便打印
- ✅ **分类管理**：算法、系统设计、行为面试、项目经验等
- ✅ **打印功能**：一键打印重点准备材料

## 使用步骤

### 1. 打开准备清单

1. 进入"求职-面试-申请列表"页面
2. 选择一个职位
3. 点击"准备材料"tab
4. 点击某个面试阶段的"准备清单"按钮

### 2. 添加清单项

点击modal右上角的"+ 添加清单项"按钮，填写：

- **清单项内容** (必填)：例如 "复习数组相关算法题"
- **分类**：算法、系统设计、行为面试、项目经验、技术栈、其他
- **标记为重点**：勾选后会出现在"重点准备项"区域
- **备注/链接**：支持Markdown格式

### 3. 添加链接示例

在"备注/链接"字段使用Markdown格式：

```markdown
- LeetCode题目: [Two Sum](https://leetcode.com/problems/two-sum/)
- 学习资料: [Hash Table详解](https://example.com/hashtable)
- 复习笔记: [我的Notion笔记](https://notion.so/my-notes)
```

保存后，链接会自动渲染为可点击的超链接（新标签页打开）。

### 4. 编辑清单项

点击清单项右侧的"编辑"按钮，可以修改：
- 清单项内容
- 分类
- 优先级
- 备注/链接
- 排序顺序

### 5. 删除清单项

点击"删除"按钮，确认后即可删除。

### 6. 打印重点材料

如果有标记为"重点"的清单项：
1. 点击"打印重点材料"按钮
2. 浏览器会打开打印预览窗口
3. 包含所有重点项的内容、分类、备注和链接

## 实际使用场景

### 场景1：准备算法面试

```markdown
清单项: 复习动态规划专题
分类: 算法
重点: ✅
备注:
- [Climbing Stairs](https://leetcode.com/problems/climbing-stairs/)
- [House Robber](https://leetcode.com/problems/house-robber/)
- [Longest Increasing Subsequence](https://leetcode.com/problems/longest-increasing-subsequence/)
- 复习笔记: [DP解题模板](https://notion.so/my-dp-notes)
```

### 场景2：准备系统设计面试

```markdown
清单项: 学习分布式系统设计
分类: 系统设计
重点: ✅
备注:
- [System Design Primer](https://github.com/donnemartin/system-design-primer)
- [设计TinyURL](https://example.com/design-tinyurl)
- [CAP定理详解](https://example.com/cap-theorem)
```

### 场景3：准备行为面试

```markdown
清单项: 准备STAR案例 - 项目冲突解决
分类: 行为面试
重点: ✅
备注:
情境: 团队成员对技术选型有分歧
任务: 需要在deadline前做出决策
行动: 组织技术评审会议，列出各方案优缺点
结果: 达成共识，按时交付项目

参考资料: [行为面试技巧](https://example.com/behavioral-interview)
```

## Tips

1. **善用Markdown链接**：将所有学习资料、练习题、笔记链接集中管理
2. **合理使用重点标记**：不要所有项目都标记为重点，聚焦最重要的5-10项
3. **按分类整理**：便于按面试轮次（算法轮、系统设计轮等）准备
4. **及时更新**：面试结束后根据反馈更新准备清单

## 数据存储

- 清单项数据存储在`interview_preparation_checklist`表
- 关联到`interview_stages`表（面试阶段）
- 每个用户只能查看和编辑自己的清单项
