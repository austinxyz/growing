## 🎓 学习路径建议

### 路径 1: 按大类系统学习（适合全面掌握）

**第1阶段: 数据结构基础**（约2-3周）
```
1. Introduction 介绍 (FA 14) - 入门必读
2. Stack 基础 (FA 37) - 数据结构原理

   - Two Pointers
   - Sliding Window (Variable + Fixed)
   - Prefix Sum
   - Binary Search
4. Linked List (FA 36)
5. Intervals (FA 41)
```

**第2阶段: 树和图**（约2-3周）
```
1. Binary Tree - DFS (FA 39):
   - Introduction → Fundamentals → Return Values
   - Passing Values & Helper Functions
   - Types of DFS
2. Graph - DFS (FA 39):
   - Graphs Overview
   - Adjacency List
3. Graph - 拓扑排序 (FA 40)
```

**第3阶段: 搜索算法**（约1-2周）
```
1. Backtracking (FA 43):
   - 概述 → Solution Space Trees
2. BFS (FA 44):
   - Introduction → Fundamentals → Graphs Overview
```

**第4阶段: 高级数据结构与动规**（约2周）
```
1. Data Structure Design (FA 42):
   - Heap
   - Trie
2. Dynamic Programming (FA 45):
   - Fundamentals → Solving a Question
3. Greedy (FA 50)
```
